## 基础功能

- 第一时间看到老师最新发布的通知及其附件

- 预览或下载课程文件，将文件分享到其他应用

- 一览作业要求，不错过任何 deadline，实时查询作业提交与批改情况

## 高级功能

- Deadline 系统日历同步

- 滑动卡片以置顶

- 隐藏 / 屏蔽课程

- 模糊搜索

## 即将加入的新功能

- 推送通知
